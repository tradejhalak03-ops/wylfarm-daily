## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit [http://localhost:8080](http://localhost:8080) to view the site.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run test` | Run tests |

## Deploy

Build the project with `npm run build`. Deploy the `dist` folder to your hosting provider (Vercel, Netlify, etc.).

**Note:** For social sharing (og:image, twitter:image), update the meta tags in `index.html` with your full deployment URL when you go live.
